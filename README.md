## ✨ 功能清单
- 🌤️ 成都实时天气查询
- 🏫 校园空教室查询
- 🗺️ 川大校园地图导航
- 👤 个人中心/登录功能
- 📢 校园公告栏（带跳转详情）

## 🛠️ 技术栈
- 微信小程序原生开发
- 免费第三方天气API
- 校园工具集成

## 📱 运行方式
1. 克隆项目到本地
2. 微信开发者工具打开项目
3. 勾选「不校验合法域名」即可运行
